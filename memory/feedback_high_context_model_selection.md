# Use Strong Models for High-Context Review

Some work is precision-bound rather than speed-bound. Legal, policy, privacy, security, consent, data-retention, and compliance reviews often require reasoning across multiple long documents, implementation files, release notes, and jurisdiction-specific references.

For those tasks, route to the strongest available long-context model with high reasoning. Do not downshift to a smaller or cheaper model when cross-document consistency, exact citations, or regulatory language precision matters.

If the preferred long-context model is unavailable, escalate or fix the runtime/account configuration instead of silently reducing model quality.
